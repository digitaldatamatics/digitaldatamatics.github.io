Usage

Importâ€‹

import {
  setServerCache,
  getServerCache,
  ServerCacheCls,
} from "path/to/server-cache/server-cache";

Pass configâ€‹

//syntax
setServerCache(options);

//example
setServerCache({
  enableRedisCache: true, 
  redisDatabaseOptions: {
    dbIPAddress: "127.0.0.1", 
    dbPort: 6379, 
    dbPassword: "password", 
  },
  keyPrefix: "MyProductName", 
  keySuffix: "Test", 
});
ParamTypeDefaultDescription
[enableRedisCache]booleanfalsetrue to use redis as server cache
[redisDatabaseOptions.dbIPAddress]string"127.0.0.1"redis server ip address
[redisDatabaseOptions.dbPort]number6379redis server port
[redisDatabaseOptions.dbPassword]string""redis server auth password
[keyPrefix]string""(optional) prefix to add against each key
[keySuffix]string""(optional) suffix to add against each key

Get Instanceâ€‹

const serverCacheInst: ServerCacheCls = getServerCache();

Use methodsâ€‹

Note : Can also check unit tests for working samples (server-cache/test/server-cache.methods.ts)

Set Getâ€‹


const key = "name"; //string
const value = "Alex"; //can be string, number, boolean, date, object, stringArray, numberArray, ObjectArray

await serverCacheInst.set(key, value);

const cacheValue = await serverCacheInst.get(_key);

Multiple Set Getâ€‹

Sets the given values to their respective keys


//multiple set
const valueObj = {
    fatherName: "Dad",
    fatherAge: 50
    fatherDob: new Date("1972-01-01"),
};
await serverCacheInst.multipleSet(valueObj);

//single get
const fatherName = await serverCacheInst.get("fatherName");
const fatherAge = await serverCacheInst.get("fatherAge");

//multiple get
const keyObj = {
    fatherName: 1,
    fatherAge: 1,
};
const multiValObj = await serverCacheInst.multipleGet(keyObj);


Deleteâ€‹

//single delete
const key = "name";
await serverCacheInst.delete(key);

//multiple delete
const keyObj = {
    fatherName: 1,
    name: 1,
};

const deleteCount = await serverCacheInst.multipleDelete(keyObj);

Check key existsâ€‹

//single exists
const key = "name";
const count =  await serverCacheInst.exists(key); //count == 1

//multiple exists
const keyObj = {
    fatherName: 1,
    name: 1,
};

const keysCount = await serverCacheInst.multipleExists(keyObj); //keysCount == Object.keys(keyObj).length

expiryâ€‹


const key = "bestInvestment";
const value = "learning";
const expireAfterSeconds = 1;

//setting expiry while setting value
await serverCacheInst.set(key, value, expireAfterSeconds);

//setting expiry later
await serverCacheInst.setExpiry(key, expireAfterSeconds);

//check remaining expiry time
const remainingTime = await serverCacheInst.getExpiryRemaining(key);

//remove expiry
await serverCacheInst.removeExpiry(key);

keysâ€‹


//get total key count in database
const totalCount = await serverCacheInst.getTotalKeyCount();

//get all key names
const totalKeysObj = await serverCacheInst.getKeyNames();

//get specific key names by pattern
const pattern = "**user*"; //keys containing "user" word
const patternKeysObj = await serverCacheInst.getKeyNames(pattern);

//to get values of those keys
const multiValObj = await serverCacheInst.multipleGet(patternKeysObj);

customâ€‹

To use raw library methods (useful when above methods doesn't solve our requirement)


import {
    Redis //just for types (typescript intellisense of raw ioredis)
} from "path/to/server-cache/dependencies"; 


//@ts-ignore
const ioredisInstance: Redis.Redis = serverCacheInst.libInst;

//can use raw ioredis methods
await ioredisInstance.set(key, value);
